**JStarCraft Cloud是一个面向中间件与云服务的轻量级框架,遵循Apache 2.0协议.**

目标是提供一个通用的中间件抽象层,作为微服务项目的基础.

使得研发人员能够在各种中间件实现层之间无缝切换.

涵盖了网关,治理,链路和配置4个方面.
